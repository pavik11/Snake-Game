# Sprint 6 Snake Game

Sprint 6 is a simple Snake game built with Pygame. The player controls the snake with the arrow keys, collects apples to increase the score, and avoids poison berries and walls.

## How to Run

1. Make sure Python and Pygame are installed.
2. Open a terminal in the `1.2.5` folder.
3. Run:

```bash
python sprint_6.py
```

## Controls

- `Up Arrow` moves the snake up
- `Down Arrow` moves the snake down
- `Left Arrow` moves the snake left
- `Right Arrow` moves the snake right

## Game Rules

- Eating an apple increases the score.
- Eating a berry decreases the score.
- The game ends if the snake hits a wall.
- The game also ends if the score drops below 0.

## Files Used

- `sprint_6.py` - main game file
- `apple.png` - collectible item that increases score
- `berry.png` - penalty item that decreases score
